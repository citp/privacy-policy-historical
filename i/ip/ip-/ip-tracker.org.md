# Privacy Policy | IP-Tracker.org

### Cookies

Cookies are small text based information packets stored by the web based browsers (such as Chrome, Internet Explorer, Firefox or any other web based browser) on user computer. The information is then sent back to the server each time the browser requests a page from the server. This enables a web server to identify and track web browsers. 

### Google Analytics Cookie

Google Analytics generates statistical and other information about website use by means of cookies, which are stored on user computers. IP-Tracker.org use Google Analytics to analyse and collect information how do you use IP-Tracker.org. The information generated relating to our website is used to create reports about the use of the website. IP-Tracker.org don't collect or store your personal information (for example your name or physical address) so information from Google Analytics can't be used to identify who you are. 

We don't allow Google to use or share our analytics data. 

To opt out Google Analytics Cookie visit [Google Analytics Opt-out Browser Add-on](https://tools.google.com/dlpage/gaoptout)

If you do not want to opt out then please visit [Google Ads Settings](https://www.google.com/settings/ads/) to edit the preferences
