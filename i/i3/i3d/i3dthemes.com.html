A web template is a package of files, normally pre-built as a ready-to-go website, that typically are edited in a web editor application such as Dreamweaver, Expression Web, Sharepoint Designer, or FrontPage.

A web template from us is designed to be easily updated without much need for additional software above and beyond the web editor.

Most web templates include a few different layouts, some with upwards of ten or more pre-designed pages such as the About Us and Contact Us pages.
A web editor is an application, either installed on your computer, or run through your web browser, that is capable of editing modern web editing components such as global Library/Include files, and Dynamic Web Templates (DWTs).

Typically in the range of $100 to $400, the cost involved in purchasing a web editor is quickly recouped usually over the lifespan of a single project, when considering editing a website with only a text editor could take 10 times as long to complete compared to if someone used a modern web editor.

Current modern web editors used in the market include: FrontPage 2003, Expression Web, Sharepoint Designer 2007, Dreamweaver
Typically the only software you will need to edit one of our current generation products is a web editor such as:
  • FrontPage 2003
  • Expression Web 1/2/3
  • Sharepoint Designer 2007
  • Dreamweaver CS3/CS4
Apart from a web editor, you may need photo manipulation software to resize your images. You can use the online service called PicNik.com (for free) to make simple modifications to photos.

In Generation 8 (and earlier) templates, if you wish to edit the "text logo" yourself, you may need Adobe Fireworks and Adobe Flash (and sometimes SwishMax) -- we offer to update the text logo in these templates for no charge.

In the latest generation of web teplate (Gen 9), you do not need any additional software to update the text logo.
Unfortunately, we do not have the geek-power available, at this time, to provide you with a "webmastering" option.
Microsoft FrontPage is no longer available for sale.

Microsoft discontinued sale and support for their FrontPage line of web authoring software at the end of 2006 when the launched their Expression and Sharepoint line of web authoring applications.

We recommend either Expression Web or Sharepoint Designer as a replacement for FrontPage.
Yes, everywhere you see a "photo" of some sort or another, such as in the "flash", you can change yourself.

You do not need additional software (such as flash) -- it is as simple as dragging new photos into the template.

You may need to resize the images you intend to use, to match the size of those used in the template. If you do not have photo editing software, you can use picnik.com (a free photo editing website).
Source files are provided for web designers. If you have a professional level of experience with the latest versions of Adobe Fireworks and Adobe Flash, you can modify the .fla and .png source files yourself. There is no support or instruction, however, provide for the modification of such items.

To give you an idea of what is involved, it takes our designers upwards of a couple of hours to colorize a template. Considering they're used to the software, and the templates, it could take someone who isn't experienced, considerably longer.

We recommend that you find a template that you like the layout and the color of.

If you're set on a template that isn't in the color choice you're interested in, give us a call and we can look into other options.
