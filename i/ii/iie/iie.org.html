Site Visits:
Whenever you request a page through your browser, the navigation and clickstream data such as IP address, browser (and version), operating system, date, time, and the site from which you came are stored in a log file. IIE uses software programs to create summaries from these statistics for such purposes as evaluating content worth, Website and system administration, and anonymous user analysis. This information cannot be used to identify specific individuals.

Cookies:
IIE does not use cookies.

My IIE:
The information you submit when signing up for My IIE is completely voluntary. The information that is collected is only used to enhance your IIE.org experience with customized content. IIE.org stores that information in a database and does not in any way sell, market, or distribute the submitted information.
