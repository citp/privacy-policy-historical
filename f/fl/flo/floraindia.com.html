How we use "Cookies" and "Action Tags".

A "cookie" is a small data file sent to your browser from a web server and stored on your computer. We use cookies to let us know that you are a prior customer, user or Member and to provide certain features to you. Most web browsers allow you to instruct the browser to prevent the use of cookies. Our Web Site also uses a common web technology "called "action tags" (also called single pixel gifs or web beacons) to anonymously track purchase, usage and other web activity information.
