This product is a media player which supports FLV, F4V, MP4 and MOV video formats. It plays videos in HD and Full Screen. You can create your own personal playlists. It is an easy-to-use and intuitive application.

## 5\. Instructions to uninstall the software from this web:

  1. Go to (Start)-> (Computer)
  2. Uninstall or change a program
  3. Select FLV Media Player version 1.3
  4. Click on Remove (Uninstall)
  5. Click 'Yes' to confirm the Uninstall
  6. And finally, when the installation is done, we can see a message saying that the uninstall process was satisfactory. Click on 'OK'
