Special terms for Software "MyPhoneExplorer"

MyPhoneExplorer Client (the application on the Smartphone) does not connect to Internet and it does send data ONLY to MyPhoneExplorer Desktop which is installed on PC. MyPhoneExplorer Desktop does connect to the internet only in  this use-cases:

1)  update check on application launch, no private data are transmitted
2)  Name/Address search based on telephone number: The request gets cached for two weeks due technical reasons. Your IP-Address gets logged to prevent mis-usage, but it’s not saved permanently.

Besides this MyPhoneExplorer does not transmit any private data to the internet.
