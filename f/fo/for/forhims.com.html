Scripts are sometimes used to collect information about the users interactions with the website, such as the links the user clicks on. Scripts are often times temporarily downloaded to the users computer or device from the website server, active only while the user is connected to the Site and/or the App, and deactivated or deleted when the user disconnects from the website.
