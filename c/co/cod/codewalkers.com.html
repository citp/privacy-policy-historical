Recommended PHP Book

Are you looking for an excellent book on PHP? Codewalkers heartily recommends 'Beginning PHP and MySQL 5: From Novice to Professional, Second Edition' by W. Jason Gilmore. The technical reviewer for the book was Matt Wade, owner of this site.

In this book, you will find coverage of PHP5 and MySQL 5. Most of what is in the book will also work for PHP4 and MySQL4 as well. The book truly takes you from Beginner to Professional, so if you find yourself with a need to learn PHP...this is the book for you.

codewalker /n./ A program component that traverses other programs for a living. Compilers have codewalkers in their front ends; so do cross-reference generators and some database front ends. Other utility programs that try to do too much with source code may turn into codewalkers. As in "This new `vgrind' feature would require a codewalker to implement."
