# Privacy Policy: California Digital Library

The CDL is committed to protecting your privacy. The following Privacy Policy describes what information we collect from you when you visit this site, and how we use this information.

## The Information CDL Collects

The CDL gateway site, [http://www.cdlib.org](http://www.cdlib.org/), collects information to improve functionality and content and to monitor performance.  Data is used to help answer specific questions about the usage and performance of the web site or individual web pages.

CDL collects tracking information that is automatically collected as visitors navigate through this site.  This site automatically recognizes and records certain non-personal information, including the name of the domain and host from which you access the Internet (for example, aol.com or ucop.edu); the Internet protocol (IP) address of the computer you are using; the browser software you use and the operating system; the date and time you access this site; and the Internet address of the web site from which you linked directly to this site.  At no time is site usage associated with individual IP addresses.

## Your Consent

By using this web site, you consent to the collection and use of the information described above by CDL.  We do not sell, trade, or rent the information gathered to others.
