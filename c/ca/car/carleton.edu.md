# Carleton College: Privacy Policy

**Use of Information**  
We analyze information collected by WebTrends Live to improve the content and functionality of our web siteï¿½to better serve our users. Carleton does not seek personally identifiable information (e.g. name, email address) nor does it provide or sell any personally identifiable information to outside agencies.

When you visit the Carleton web site, the following information maybe gathered: 

  * Your Internet location (IP address)
  * Which pages you visit on our site
  * From which page you came to our site
  * Which software you use to visit our site and its configuration



**WebTrends Live**  
WebTrends Live does not create individual profiles for visitors. Unlike some tracking services WebTrends Live does not have a database of individual profiles for each visitor. WebTrends Live only collects aggregate data. For more information about WebTrends Live privacy policy, please click here: <http://www.webtrendslive.com/privacy_policy.htm>

**Disclaimer**  
The Carleton web site provides links to web sites outside the Carleton.edu domain. The College web site cannot be responsible for the privacy practices or the content of such web sites.

If you have questions about the Carleton web site, or about the policies of this site, please contact our [Web Producer/Designer](mailto:bkoranda@acs.carleton.edu).
