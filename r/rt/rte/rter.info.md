# RTER.info Privacy

User privacy is a fairly careful and rigorous things for us. On RTER.info you can use all the features, without any registering and login. We currently uses [Piwik](http://piwik.org/)(OpenSource) as site traffic statistics, functions as Google Analytics or Yahoo Webmaster tools. The type of information collected as follows, and this information for statistical use of RTER.info will not be provided to any third party.

### **Data collection**

We will collected some information Include IP Address, browser information, pages viewed, use for improve the content of our web presentation, layout, in order to provide users with a better browsing effect.
