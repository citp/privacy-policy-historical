Search Tips
Search within a category?
You can search for projects within category as follows:
Type 'category: <search string>' in the search box.
Example, 'solar:remote control' will search for all the projects with string 'remote control' in the project name & abstract in the 'Solar' category
Search for a model number?
You can search for a model number as follows:
Type 'Model:<model number>' in the search box.
Example, 'Model:171' will search for the project with model number 171
Search for a project?
You can search for a project as follows:
Type '<search string>' in the search box.
Example, 'wireless' will search for all the projects with string 'wireless' in the project name & abstract
