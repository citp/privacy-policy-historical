GeekPolice will collect personally identifiable information (e.g., name, e-mail address, postal address) from you when you voluntarily provide it. II. AGGREGATE INFORMATION COLLECTED

Aggregate information is non-personally identifiable or anonymous information, such as type of web browser used, type of operating system, the domain name of your Internet Service Provider (ISP), and Internet Protocol (IP) addresses. Each time a visitor (whether they are registered or not) connects to a page on this website, we log that visitor's IP address in our web server's log files. This is a common practice used by almost all sites and is used to track which pages visitors access and how much time is spent on a page. GeekPolice also uses cookies to enhance user experience on this website. Cookies are small text files that a web site sets on your computer's hard drive in order to recognize you in the future. Cookies make using our Site easier, as they allow you to register an account, stay logged in as you browse this website, and save your password for future visits. Third parties may use cookies as described in Section 4 below. Sometimes cookies are substituted by session data set by a web programming language called JavaScript, to which similar data is stored, and usually erased as you exit the website. You may disable cookies any time. However, if you do not allow cookies from GeekPolice, then you may not be able to use certain features of this website. Certain web pages may contain invisible images, sometimes called web beacons (a 1px GIF file usually) that allow a web site to count users who have visited that page. GeekPolice will only use web beacons within this website for statistical purposes. Third parties may use web beacons as described below.
