# Privacy Policy | Game Tabs // Video Game Tablature

Game Tabs only collects IP address information from all users who visit the site, but does not use that information for anything other than general record keeping.

Game Tabs uses cookie tracking technology for users who wish to register an account with the site and use the features that having an account allows. No cookie information is kept for guest users.

The only situation in which we would share personal information about our users is if we were contacted by a government agency and ordered to do so.

All information on this site is kept secure through up to date coding standards.
