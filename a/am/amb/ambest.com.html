Cookies
A cookie is a piece of data stored on the user's computer tied to information about the user. We use both session ID cookies and persistent cookies. For the session ID cookie, once users close the browser, the cookie simply terminates. A persistent cookie is a small text file stored on the user's hard drive for an extended period of time. Persistent cookies can be removed by following Internet browser help file directions.

For more information on cookies, visit Microsoft's site if you are using Internet Explorer. Or visit NetScape's site if you are using Netscape Navigator as your browser.

Some of A.M. Best's pages use "cookies" to monitor your activity throughout the sites and correlate usage patterns. By setting a cookie on the Web Site, users do not have to log in every time they visit, thereby saving time while on the Web Site. If users reject the cookie, they may still use the Web Site. The only drawback to this is that the user will be limited in some areas of the Web Site.

For example, if a user rejects cookies, they will constantly be asked to verify their access every time they wished to access a Best's Rating. The persistent cookies mentioned earlier enable us to track and target the interests of our users to enhance the experience on the Web Site.
