Manage Accounts

The Manage Accounts page is where you can manage your parking service accounts.

Service
This column shows the parking service name.

Account
This column shows the username of this parking service's account.
Click a username to view daily revenue statistics for that account.

Enabled
This column lets you know if this account is enabled or disabled in Above.com.
To change the setting, click Edit.

Edit
Clicking Edit will allow you to change settings for this account or remove this parking service account from Above.com.

See the Manage Accounts manual page for more information.

Dashboard

This table shows an overview of daily statistics from the last 10 days.
Incomplete days and days with estimated values are in red and will update when final results are available.
You can click the icon to show more.

Day
Click a date in this column for a detailed break down of that day.

Hits
Total hits (page impressions) for all domains for this day.

Uniques
Total unique hits for all domains for this day. This value is only available when domains are using the Above.com DNS.

Reported
Total unique hits for all domains for this day, reported by parking companies.

Clicks
The total number of clicks reported for this day.

CTR
The average click through ratio, calculated from reported uniques.

nCTR
The normalised average click through ratio. This value is calculated from internal statistics and is only valid if ALL domains in the account are using the Above.com DNS.

EPC
The average earning per click.

RPM
Average revenue per thousand unique visitors.

nRPM
A normalized RPM (revenue per thousand unique visitors) value. This value is only valid if ALL domains in the account are using the Above.com DNS.

Revenue
Total parking revenues earned for this day.
Note: Estimates are always reported in red and are updated when final results are available.

See the Dashboard manual page for more information.

Blacklist Manager

Not all services will accept traffic from all domains. To ensure that the optimizer does not send traffic to services where the traffic can not be monetized, you can maintain a blacklist for each service.

Effective blacklist management ensures that no traffic is wasted on services where it can not be monetized.

Note: You can quickly switch between services by using the Services menu.

Tip: To view the current domain blacklist for this service, click View (or Export to export a .CSV file).

See the Managing Blacklists manual page for more information.

Domains Conflicts

The conflicts page shows domains in your account that have also been added by another Above.com user.
You can claim or delete domains in the list. However, please ensure that the domains you claim still belong to you.

Domains marked with a red exclamation icon are currently associated with another user's account, but have also been added to your account.
You can claim these domains if they belong to you, otherwise delete them.

The yellow exclamation icon indicates that the domain is claimed by you and the conflict is pending resolution.
Please insure your whois information is up to date or email support a screenshot of the domain in your registrar account to prove ownership.

See the Conflicts manual page for more information.

Maximizer

When Enabled, Above.com will utilize various strategies to maximize the performance of all domains in the folder. An example of these options include a Premium Direct to Advertiser Network and the Redirection of traffic from geographic regions typically not monetized by the parking services, such as Chinese Traffic.

By enabling this option, you authorize Above.com to employ such strategies to maximize the domain revenues.

For more information on folder settings, see Manage Folders manual page for more information.

Folder Lock

Manage Folders

The Manage Folders page allows you to change various settings for each folder, export domain lists, add / remove domains and add / remove folders.

You can export a CSV or all domains in your account by clicking the Export all domains button. To export a single folder, click the number in the Total domains column, for the folder you wish to export.

Folder
The folder name. You can click the name to edit the folder settings.
Note: The (Deleted) folder allows you to see domains deleted from your account.

Total domains
This column shows the number of domains in this folder. You can click a number to export a list of domains in the folder.

Lock
This column lets you know if the lock is turned on. If the lock is on, you will be asked before any domains are deleted or moved out of this folder.
To change the lock setting, click Edit to change folder settings.

Optimizer
This column indicates whether or not the Auto Optimizer feature is enable on this folder.
To change the Auto Optimizer setting, click Edit to change folder settings.

Service
This column tells you which parking service is being used with domains in this folder. Auto is shown if the Auto Optimizer is enabled.

Redirect
Instead of directing domain traffic for a folder to a parking service, you can redirect the traffic to another URL.
This can be useful if you want to drive traffic to a new website, without having to change your domains' DNS settings.

Maximizer
This column indicated whether or not the Maximizer is enabled.
To change the Maximizer setting, click Edit to change folder settings.

Edit or Delete
These options allow you to Edit or Delete the folder.

For more information on folder settings, see Manage Folders manual page for more information.

Jobs Queue

The Jobs Queue page shows details about statistics downloads from parking services accounts.

This log can help you or your account manager identify issues with your parking service accounts.

Date
The date of the task.

Service
The parking service that statistics were collected from.

Account
Which parking service account statistics were collected from.

Download
The date and time the download was started.

Size
Number of domains statistics were downloaded for.

Import
The date and time the the statistics were imported into our database.

Revenue
Revenue for this parking account (usually for the previous 3-4 days).

Message
If a message other that LOGIN FAILED is shown, there is no need for you to do anything. Our system will automatically retry later.

Usually SUCCESS, which means statistics were downloaded successfully.

See the Jobs Queue manual page for more information and the full list of message codes.

Missing Traffic Report

This report lists domains where traffic forwarded by Above.com has not been reported by your parking company.

This may indicate a potential problem. The possible reasons are:

  • The domain has not been added to your parking account.
  • The domain is currently in the account of another user.
  • The parking company does not recognize the traffic as valid and is not reporting it.
  • The parking company has blacklisted the domain.

See the Missing Traffic manual page for more information and what you should do.

Auto Optimizer

Above.com is designed to maximize the revenues from your domains by identifying the best place to park each and every domain name. To use this feature, the Auto Optimizer must be enabled in the folder settings.

IMPORTANT!!! If you are using the auto optimize feature, you MUST ensure that ALL domains you wish to manage have been added to all of your parking accounts.

For more information on folder settings, see Manage Folders manual page for more information.

Folder Lock

When the folder lock option is ticked, domains can not be moved out of the folder. When locked, any domain move operation will fail, or require a manual confirmation.

Custom Service & Redirect

This option allows you to direct traffic to a specific parking service, or set of parking services, custom URL, for sale page, blank or static page or any combination of these options. You can use as many parking services or combinations of any of these options as you want and define what percentage of traffic should be directed to each.

Custom 302 Redirect

This option will redirect traffic to a custom URL.

Custom Frame Redirect

Similar to Custom 302 Redirect, except the specified URL will be displayed in an HTML frame, meaning the domain name will be retained in the address bar.
