Each user’s IP address is logged with each visit and each action performed on joyofkosher.com.  This information is never provided to the third parties, except as otherwise stated in this Privacy Policy, and is maintained only for a temporary period of time.  IP address information is linked to personal information and may be used in an effort to block certain users from accessing the web site.
