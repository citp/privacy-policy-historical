1.4 Cookies, Web Beacons, and Other Info Collected Using Technology.
JobsFoundHere.com currently uses cookie and web beacon technology to associate certain Internet-related information about an individual with information about the individual in our database. Additionally, JobsFoundHere.com may use other new and evolving sources of information in the future (together, "Technology Information").
(a) Cookies.  A cookie is a small amount of data stored on the hard drive of the individual's computer that allows JobsFoundHere.com to identify the individual with his or her corresponding data that resides in JobsFoundHere.com's database. You may read more about cookies at http://cookiecentral.com. Individuals who use the Website need to accept cookies in order to use all of the features and functionality of the Website.
(b) Web Beacons.  A web beacon is programming code that can be used to display an image on a web page (by using an <img src="x"> programming function -- see http://truste.org for more information), but can also be used to transfer an individual's unique user identification (often in the form of a cookie) to a database and associate the individual with previously acquired information about an individual in a database. This allows JobsFoundHere.com to track certain web sites an individual visits online. Web beacons are used to determine products or services an individual may be interested in, and to track online behavioral habits for marketing purposes. For example, JobsFoundHere.com might place, with the consent of a Third Party website, a web beacon on the Third Party's website where baseball products are sold. When Joe, an individual listed in JobsFoundHere.com's database, visits the fishing website, JobsFoundHere.com receives notice by means of the web beacon that Joe visited the baseball site, and JobsFoundHere.com would then update Joe's profile with the information that Joe is interested in baseball. JobsFoundHere.com may thereafter present offers of baseball related products and services to Joe. In addition to using web beacons on web pages, JobsFoundHere.com also uses web beacons in email messages sent to individuals listed in JobsFoundHere.com's database.
(c) New Technology.  The use of technology on the Internet, including cookies and web beacons, is rapidly evolving, as is JobsFoundHere.com's use of new and evolving technology. As a result, JobsFoundHere.com strongly encourages individuals to revisit this policy for any updates regarding its use of technology.
