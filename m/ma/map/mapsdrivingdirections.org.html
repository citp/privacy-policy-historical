1.4 Cookies, Web Beacons, and Other Info Collected Using Technology.

mapsdrivingdirections.org currently uses cookie and web beacon technology to associate certain Internet-related information about an individual with information about the individual in our database. Additionally, mapsdrivingdirections.org may use other new and evolving sources of information in the future (together, "Technology Information").

(a) Cookies. A cookie is a small amount of data stored on the hard drive of the individual's computer that allows mapsdrivingdirections.org to identify the individual with his or her corresponding data that resides in mapsdrivingdirections.org's database. You may read more about cookies at http://cookiecentral.com

. Individuals who use the Website need to accept cookies in order to use all of the features and functionality of the Website.

(b) Web Beacons. A web beacon is programming code that can be used to display an image on a web page (by using an programming function (see http://truste.org

for more information), but can also be used to transfer an individual's unique user identification (often in the form of a cookie) to a database and associate the individual with previously acquired information about an individual in a database. This allows mapsdrivingdirections.org to track certain web sites an individual visits online. Web beacons are used to determine products or services an individual may be interested in, and to track online behavioral habits for marketing purposes. For example, mapsdrivingdirections.org might place, with the consent of a Third Party website, a web beacon on the Third Party's website where baseball products are sold. When Joe, an individual listed in mapsdrivingdirections.org's database, visits the baseball website, mapsdrivingdirections.org receives notice by means of the web beacon that Joe visited the baseball site, and mapsdrivingdirections.org would then update Joe's profile with the information that Joe is interested in baseball. mapsdrivingdirections.org may thereafter present offers of baseball related products and services to Joe. In addition to using web beacons on web pages, mapsdrivingdirections.org also uses web beacons in email messages sent to individuals listed in mapsdrivingdirections.org's database.

(c) New Technology. The use of technology on the Internet, including search engine indexing, cookies and web beacons, is rapidly evolving, as is mapsdrivingdirections.org's use of new and evolving technology. As a result, mapsdrivingdirections.org strongly encourages individuals to revisit this policy for any updates regarding its use of technology.
