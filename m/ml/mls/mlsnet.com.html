Clickstream Data. Our servers also collect clickstream data, which includes a trail of electronic information left at each web site and web page you visit, such as the referring page that linked you to us (e.g., another Web site or a search engine); the pages you visit on this Web Site; the Web site you visit after this Web Site; other information about the type of Web browser, computer, platform, related software and settings you are using; any search terms you have entered on this Web Site or a referral site; and other Web usage activity and data logged by our Web servers.We may use this information to learn about and determine how much time visitors spend on each page of the Web Site or the Service, how visitors navigate throughout the Web Site or the Service and how we may tailor our web pages to better meet the needs of visitors.
