# MediaArea - Privacy

## Personal data

MediaArea tries to collect only the minimun personal data, they are not used to track users and they are not sold to other companies.  
Here is the list of personal data we collect 

  * User account (when you [register](https://web.archive.org/register/) as a user) 
    * Email address
    * Name (not mandatory)
    * Username (not mandatory)
  * MediaConch API (when you connect [MediaConch](https://web.archive.org/MediaConch) to [MediaConchOnline](https://web.archive.org/MediaConchOnline/)) 
    * IP address
  * Payment system (when you made a payment to [support us](https://web.archive.org/SupportUs/Individual)) 
    * IP address

If you want to retrieve all the data we have about you or if you want to delete your account you can [contact us](https://web.archive.org/Contact). 

## Cookies

MediaArea uses [Google AdSense](https://www.google.com/adsense/start/) to display ads (you can avoid ads by [supporting us](https://web.archive.org/SupportUs/Individual)), this service use cookies.
