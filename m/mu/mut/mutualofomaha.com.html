Mutual places Internet cookies on our Users' hard drives. A cookie is a small file that is sent from Mutual's Web server to a Users' personal computer. This file contains information about a User's visit to Mutual's Web site that only Mutual's server can read. The cookies are stored on the User's hard drive and are not stored, in any form, by Mutual. Information derived from reading the cookie is stored by Mutual and is protected under the terms of Mutual's corporate data privacy/security policy. Cookies are used to enhance a User's return visit to Mutual's Web site. Users may decline the cookies by using the appropriate feature of their Web browser software, but by doing so, may experience a decrease in functionality. When applicable, cookies are used to verify the User's authorization to access certain Mutual Web pages (e.g., a broker's access to the "Sales Professional Access" portion of the Web site).
