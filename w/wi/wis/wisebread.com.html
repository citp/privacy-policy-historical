What is a Cookie?

Cookies are a feature of web browser software that allows web servers to recognize the computer used to access a website. They are small pieces of data that are stored by a user’s web browser on one website to simplify subsequent interactions with that website by the same user or to use the information to streamline the user’s transaction on related web pages. This makes it easier for a user to move from website to website and to complete transactions over the Internet. Cookies should make your online experience easier and more personalized.
