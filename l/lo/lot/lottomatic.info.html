Usage Statistics

In order to generate usage statistics to help us improve this website, we automatically collect certain information on every visitor to this website. This information does not disclose "personal identifiers" (such as the name, address, Email address, phone number, etc.). Every time a page of this website is requested, a log file contianing generic user information is being created or updated. The information contained by the log file includes the type of Internet browser or computer operating system used to make the request, the date and time the page was requested, the Internet protocol address (or proxy server) of the individual computer that made the request, and the Internet address of the web page from which the visitor may have linked to this site.
