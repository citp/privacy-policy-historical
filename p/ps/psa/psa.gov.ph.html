Privacy Policy

 
The purpose of this policy is to inform our clients on how the data are handled when they visit this site. This policy is subject to change at any time and its revision will be posted on this site.
Security of Information
 
Users should be aware that there are inherent risks associated with the transmission of information via the Internet. An e-mail, for example, is not usually secured against interception and virus infection. Its security depends on the software that is being used.
 
The data request and feedback forms that the PSA website provides do not use any encryption/decryption protocol for online transmission. Its security also depends on the browser that is being used.
 
PSA Data Request and Feedback Forms
 
Information sent thru the PSA data request and feedback forms will be used only to help us gather/process the data that maybe requested and to respond to request via e-mail. In responding to request, the information provided may be viewed by various people within the PSA. PSA personnel usually collect identifiable information such as name or e-mail address for record purposes. Nevertheless, this information will not be disclosed outside of PSA without user's consent, except when authorized or required by law.
 
Cookies
 
A cookie is a small text file that is stored on a user's computer for record-keeping purposes. The PSA website uses cookies for its tree navigation. This is to remember the users current selection in the navigation.
 
Data Collected When Visiting This Website
 
When visiting this website, the PSA web server makes a record of the user's visit and logs the following information:
  • The date and time of visit to this site
  • The ISP / location
  • The type of browser used
  • The plug-ins, operating system and resolution used
  • The previous site visited
The data listed above are collected for the following purposes:
  • to analyze trends (i.e. Generating Web site statistics);
  • to administer the site;
  • to track users' movements around the site; and
  • to gather demographic information about our user base as a whole.
 
Contact Us 
 
We reserve the right to modify this privacy statement at any time, so please review it frequently. If you have any questions or suggestions regarding our privacy policy, please contact us by phone at (632) 4626600 loc 839 or email us info@psa.gov.ph.
