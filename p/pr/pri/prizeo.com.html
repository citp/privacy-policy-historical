    10. GLOSSARY OF TERMS USED IN THIS PRIVACY POLICY
      1. Cookie: A small bit of code place on your computer by a website or advertiser as a reminder to tell the site a website or advertiser when a viewer is someone who has visited/viewed before. In technical terms, a cookie is a small electronic file containing a string of characters that a server places on your computer when you visit a site or view an ad.
      2. Internet protocol address: An individual "name" that is assigned to each computer on the Internet, so that each computer can be uniquely identified.
      3. Referring website address: The domain name or URL of the webpage you were visiting immediately prior to when you accessed, visited or used the Site.
      4. Browser: The software application on your computer that lets you "see" websites. Common browsers are Microsoft Internet Explorer, Mozilla Firefox, Google Chrome, Opera and Apple Safari.
      5. Web Beacon or Pixel: A small, transparent image that is embedded in a web page or e-mail and is usually invisible to the user but tells the web beacon owner that a user has viewed the web page or e-mail.
