PromoJunkie may also collect information about how end-users use our web site, for example, by tracking the number of unique views received by the pages of the website or the domains from which users originate. We may use "cookies" to track how Users use our Web site. A cookie is a piece of software that a Web server can store on the Users' PC and use to identify the User should they visit the Web site again. While not all of the information that we collect from Users is personally identifiable, it may be associated with personally identifiable information that Users provide us through our Web site.
